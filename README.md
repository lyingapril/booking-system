# 时间预约系统

一个基于Node.js + Express + Vue的时间预约系统。

## 功能特点

- 服务预约功能
- 预约记录查询
- 预约取消功能
- 服务管理

## 安装与启动

### 自动安装（推荐）
运行 setup-project.js 脚本会自动创建项目并安装依赖

### 手动启动

```bash
# 启动后端
cd backend
npm run dev

# 启动前端
cd ../frontend
npm run serve
```

## 技术栈

- 后端: Node.js, Express, Sequelize, SQLite
- 前端: Vue 3, Element Plus, Axios
